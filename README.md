# Twitter 1Click Block & Mute

Twitter(X)のタイムラインやプロフィールに、ワンクリックでブロック・ミュートできるボタンを追加するブラウザ拡張機能です。

## 機能

### ブロック
見たくないユーザーをワンクリックで排除します。ブロックされたユーザーのツイートはタイムラインに表示されなくなり、あなたのプロフィールも閲覧できなくなります。

### ミュート
フォローしているけどタイムラインには表示したくないユーザーをワンクリックで非表示にします。フォロー関係はそのまま維持されますが、そのユーザーのツイートはTLに流れなくなります。

## ボタンの表示場所

- タイムラインのツイート
- プロフィールページ
- おすすめユーザー(You might like)
- フォロー/フォロワー一覧
- 引用ツイート
- ホバーカード

## インストール

### Chrome

### Firefox
[Firefox Add-ons](https://addons.mozilla.org/) からインストールできます。
